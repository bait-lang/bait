Usage:
  bait [options] <command|file>

Examples:
  bait my_file.bt
  bait -o lib.js src/

Commands:
  test <dir>  Run all tests in the given directory.
  ast <file>  Prints the Abstract Syntax Tree (AST) for the given file.
  self        Recompile bait.
  up          Update to the latest git commit.
  help        Prints this message.

Options:
  -o, --out <file>   Output file name (default: "out.js").
